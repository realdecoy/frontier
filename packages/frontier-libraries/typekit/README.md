## Installation

```sh
$ yarn add --save--dev @rdfrontier/typekit
```

or 

```sh
$ npm install --save--dev @rdfrontier/typekit
```

&nbsp;
&nbsp;

## Usage

```sh
import { SuccessCode } from '@rdfrontier/typekit';

console.log(SuccessCode.Countinue)
```
